О методе генерации

Для генерации решётки используются существующие атомы и конфигурации из базы данных. Параметр решётки (a) берётся из свойств выбранного атома и преобразуется в метры (Å → м). Если у атома не указана кристаллическая структура, используется выбранная в форме.

Используемые алгоритмы:
• CrystalLatticeStrategy для генерации позиций атомов
• LatticePhysics для расчета структурных факторов
• Преобразование единиц: a = aatom × 10⁻¹⁰ м
